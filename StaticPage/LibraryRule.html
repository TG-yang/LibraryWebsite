Library Rule

About Borrowing Requirements
  1. In our library, borrower can only borrow 3 books at the same time.
  2. Normal borrowing will not have to pay any fees.
  3. If you fail to return the book after the deadline, pay more than one dollar each day.
  4. If you damage the book, the administrator will fine according to the situation.
  5. If your account balance is insufficient, you will not be able to borrow books.

About Library
  1. The library opening hours may vary from day to day. Please browse the home library information in time.
  2. Clemson University has a lot of libraries, please make sure that borrowing and returning books are in same library.
  3. Please make sure to return all books before graduation and make sure your account balance is greater than 0.
