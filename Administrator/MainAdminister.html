What should you do as a librarian?

  1. If the library business hours change, please update it.
  2. Punish people according to the damage of the book when people return books
  3. The fine is up to the price of the book, the minimum is 1 dollar
  4. When Borrower borrows books, please proceed with this process according to the ISBN of the book.
  5. If the book is damaged and there are no extra copies, please delete the book.
  6. According to the position of the book in the library after the people return the book, the administrator should put the book in the designated position in time.

How to backup and recover library system database?

  1. At first, you can click recover button to recover the database to the most recent backup.
  2. We recommend that you make a backup when there is a major change in the library.
  3. When you click backup button, the data will be stored as .csv file and those files will be saved in the server.
